Esta es la primera pagina y aqui hablaremos de dos componentes de Bootstrap

Los botones en Bootstrap se crean con la clase .btn, y pueden tener diferentes estilos según el color y contexto (.btn-primary, .btn-secondary, .btn-danger, etc.). Se usan para activar acciones como enviar formularios, abrir modales, o simplemente redirigir a otras páginas.

Las alertas en Bootstrap (.alert) sirven para mostrar mensajes informativos, de error, éxito o advertencia. Son útiles para dar retroalimentación visual a los usuarios sobre el estado de alguna acción dentro de la página. Se pueden cerrar agregando la clase .alert-dismissible junto con un botón de cierre.
